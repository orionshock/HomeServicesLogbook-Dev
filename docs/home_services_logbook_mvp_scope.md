# Home Services Logbook - MVP Scope

## Purpose

Defines the minimum viable product scope for the project.

Core concept: Vendor → Entries → Attachments

## Core Features

-   Vendor management
-   Entry logbook
-   Attachments
-   Vendor timeline
-   Simple calendar (.ics) export

## Out of Scope

-   tagging systems
-   advanced search
-   authentication
-   Home Assistant integration
-   mobile apps
-   analytics dashboards

## Success Criteria

Users can: 1. create vendors 2. write notes about interactions 3. upload
documents 4. review history later
